# Bổ sung hệ thống file và trình soạn thảo UNIX: nano and vim (cơ bản).

## Bảng tóm tắt - câu lệnh
Các lệnh cơ bản:
- `ls`: hiển thị nội dung thư mục.
- `cd`: chuyển thư mục
- `pwd`: cho biết vị trí thư mục hiện hành.
- `rm`: xóa file.
- `rmdir`: xóa thư mục.
- `cp`: sao chép file.
- `mv`: di chuyển file.
